# 1. 자율주행과 지도 학습의 개요

## 📌 개요

자율주행(Autonomous Driving)은 다양한 센서, 제어 시스템, 인공지능 기술을 종합적으로 활용하여 사람의 개입 없이 차량이 스스로 주행할 수 있도록 하는 기술이다. 이러한 시스템의 핵심 구성 요소 중 하나가 바로 **지도 학습(Supervised Learning)**이다. 지도 학습은 자율주행 차량이 환경을 인식하고, 객체를 분류하며, 행동을 결정하는 데 있어 매우 중요한 역할을 한다.

## 🎯 지도 학습의 정의

지도 학습은 레이블이 포함된 데이터를 바탕으로 입력과 출력 사이의 함수 \( f: \mathcal{X} \rightarrow \mathcal{Y} \)를 학습하는 머신러닝의 대표적인 학습 방식이다. 자율주행에서는 다음과 같은 문제에 지도 학습이 적용된다:

- 이미지 기반 객체 분류 (예: 보행자 vs. 차량)
- 거리 예측 (회귀)
- 도로 차선 인식
- 교통표지판 분류

이를 통해 자율주행 차량은 실시간으로 주변 상황을 판단하고 안전하게 주행 결정을 내릴 수 있다.

## 🔍 자율주행 시스템 구조에서의 위치

자율주행 시스템은 일반적으로 다음과 같은 계층적 구조로 이루어져 있으며, 지도 학습은 주로 **인식(Perception)** 및 **행동 예측(Behavior Prediction)** 단계에 깊이 관여한다:

```
센서 입력
  ↓
데이터 처리 (Sensor Fusion)
  ↓
1. 인식 (Perception) ← 지도 학습의 주요 적용 영역
  - 객체 탐지 / 분류
  - 도로 인식 / 주행 가능 영역 예측

2. 위치 추정 (Localization)
  - SLAM, GPS, IMU

3. 경로 계획 (Planning)
  - 고수준 행동 결정 (예: 차선 변경)
  - 경로 최적화 알고리즘

4. 제어 (Control)
  - 차량의 물리적 조작
```

지도 학습은 특히 **고정된 태스크 정의**, **정답 데이터(label)**가 명확한 환경에서 높은 성능을 발휘하며, 인식 시스템의 정밀도를 좌우한다.

## 🧠 왜 지도 학습인가?

- **대규모 라벨링된 주행 데이터가 존재** (예: Waymo, nuScenes, KITTI)
- **해당 태스크에 대해 명확한 목표 정의 가능** (예: '신호등의 상태 분류')
- **모델 평가 및 성능 비교 용이** (정확도, Precision, Recall 등)
- **Deep Learning과 결합 시 높은 정확도 달성 가능**

## 🧩 지도 학습이 해결하는 자율주행 문제

| 영역           | 지도 학습 적용 예시                     | 출력 타입     |
|----------------|------------------------------------------|----------------|
| 객체 탐지       | 보행자/차량/자전거 분류                | 다중 클래스 분류 |
| 거리 추정       | 스테레오 영상 기반 깊이 추정            | 회귀             |
| 주행 가능 영역 | Free-space segmentation                | 픽셀 단위 분류   |
| 행동 예측       | 앞 차량의 차선 변경 여부 예측          | 이진 분류       |
